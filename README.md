# nowytest

## .NET 8 HelloWorld

Equivalent .NET implementation is available under `dotnet/` with solution file `HelloWorldDotNet.slnx`.

### Build
`dotnet build HelloWorldDotNet.slnx`

### Run
`dotnet run --project dotnet/src/HelloWorld/HelloWorld.csproj`

### Test
`dotnet test HelloWorldDotNet.slnx`
